RetroTrack Support

## Getting Started
RetroTrack requires a free RetroAchievements.org account.
Sign up at https://retroachievements.org if you don't have one.

  ## Common Issues

  ### Can't log in?
  - Verify your username and API key are correct
  - Get your API key from: https://retroachievements.org/settings → API Key
  - Make sure you're entering your username, not email

  ### Achievements not updating?
  - Pull down to refresh on any screen
  - Check your internet connection
  - RetroAchievements.org may be temporarily down

  ### Widgets not showing data?
  - Open the app and log in first
  - Widgets update every 15-30 minutes
  - Try removing and re-adding the widget

  ### Restore Premium Purchase
  - Go to Settings → Restore Purchases
  - Make sure you're signed into the same Apple ID used for purchase

  ## Privacy
  RetroTrack only connects to RetroAchievements.org using your API key.
  We do not collect or store any personal data.
